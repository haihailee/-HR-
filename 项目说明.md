# 汽车行业HR情报监控系统

## 项目简介
这是一个专门为汽车行业HR从业者打造的竞争情报监控平台，自动抓取和分析特斯拉、小米、问界、小鹏、蔚来等公司的人力资源相关新闻和行业报告。

## 核心功能
- ✅ 每日自动抓取目标公司HR相关新闻
- ✅ AI智能分类（按公司、按HR模块）
- ✅ AI自动总结关键信息
- ✅ 现代化Web界面展示
- ✅ 多维度筛选和搜索
- ✅ 完全免费，零成本运行

## 技术架构

### 前端展示
- **框架**: Streamlit（Python Web框架，零前端代码）
- **部署**: Streamlit Cloud（免费）
- **特点**: 响应式设计，支持手机访问

### 数据抓取
- **工具**: Python + requests + BeautifulSoup
- **目标网站**:
  - 各大科技媒体（36氪、钛媒体、虎嗅等）
  - 汽车垂直媒体（电动汽车观察家、汽车之家等）
  - 行业咨询网站
- **频率**: 每天自动运行

### AI分析
- **服务**: 智谱AI（免费额度）或豆包API（免费）
- **功能**:
  - 识别是否为HR相关内容
  - 分类到具体HR模块
  - 生成内容摘要

### 数据存储
- **方案1**: JSON文件（简单，适合小数据量）
- **方案2**: SQLite数据库（本地，免费）
- **方案3**: 飞书多维表格（云端同步）

### 自动化运行
- **方案1**: GitHub Actions（云端定时任务，免费）
- **方案2**: 本地电脑定时任务（需要一台常开的电脑）

## 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置API密钥
编辑 `配置文件.yaml`，填入免费AI服务的API密钥

### 3. 运行数据抓取
```bash
python 数据抓取/新闻爬虫.py
```

### 4. 启动Web界面
```bash
streamlit run 主应用.py
```

### 5. 部署到云端
按照 `部署指南.md` 的说明部署到Streamlit Cloud
