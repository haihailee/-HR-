# 快速开始 - 5分钟上手指南

## 第一次使用？看这里！

### 1️⃣ 安装Python依赖（首次运行）

打开终端，进入项目目录，运行：

```bash
cd 汽车行业HR情报监控系统
pip install -r requirements.txt
```

> 等待1-2分钟安装完成

---

### 2️⃣ 配置AI密钥（首次运行）

#### 获取免费API密钥：

1. 访问智谱AI官网：https://open.bigmodel.cn/
2. 注册并登录（需要实名认证）
3. 进入"API管理" → "创建API Key"
4. 复制密钥（只显示一次，务必保存）

#### 填入配置文件：

打开 `配置文件.yaml`，找到这一行：

```yaml
api_key: "your_zhipu_api_key_here"
```

替换为你的密钥：

```yaml
api_key: "实际的密钥字符串"
```

保存文件。

---

### 3️⃣ 运行程序

#### 方式一：使用快速启动脚本（推荐）

```bash
python 快速启动.py
```

按照菜单提示操作即可。

#### 方式二：手动逐步运行

```bash
# 步骤1: 抓取新闻
python 数据抓取/新闻爬虫.py

# 步骤2: AI分析
python AI分析/内容分类.py

# 步骤3: 启动Web界面
streamlit run 主应用.py
```

---

### 4️⃣ 访问界面

运行后，浏览器会自动打开：
```
http://localhost:8501
```

如果没有自动打开，手动在浏览器输入上面的地址。

## 常见问题

### Q: 运行快速启动.py后一直卡住？

A: 第一次抓取需要1-3分钟，请耐心等待。可以查看终端输出了解进度。

### Q: 提示"缺少依赖包"？

A: 重新运行安装命令：
```bash
pip install -r requirements.txt
```

### Q: 界面打不开？

A: 检查终端是否有报错。如果端口被占用，可以指定其他端口：
```bash
streamlit run 主应用.py --server.port 8502
```

### Q: AI分析很慢或失败？

A:
1. 检查API Key是否正确
2. 检查网络连接
3. 查看智谱AI控制台的额度使用情况

### Q: 爬虫没有抓到数据？

A: 第一次运行可能需要调试。可以先使用模拟数据测试界面。

## 自定义配置

所有配置都在 `配置文件.yaml` 中：

### 添加监控公司：
```yaml
companies:
  - name: "理想汽车"
    keywords: ["理想汽车", "理想ONE", "李想"]
    enabled: true
```

### 调整抓取频率：
```yaml
crawler:
  days_to_fetch: 7  # 改为14天抓取更多历史数据
```

### 修改HR分类：
```yaml
hr_categories:
  - name: "自定义分类"
    keywords: ["关键词1", "关键词2"]
```
