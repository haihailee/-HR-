# 项目文件结构

```
汽车行业HR情报监控系统/
│
├── 📄 快速开始.md                    # 5分钟快速上手指南 ⭐ 新手先看这个
├── 📄 项目说明.md                    # 完整的项目介绍和技术架构
├── 📄 部署指南.md                    # 详细的部署和配置教程
├── 📄 配置文件.yaml                  # 系统配置文件（需要填入API Key）
├── 📄 requirements.txt              # Python依赖包列表
├── 📄 .gitignore                    # Git忽略文件配置
│
├── 🚀 快速启动.py                    # 一键启动脚本（推荐使用）
├── 🌐 主应用.py                      # Streamlit Web界面主程序
│
├── 📁 数据抓取/
│   ├── 新闻爬虫.py                   # 新闻抓取核心模块
│   └── 数据源配置.py                 # 新闻数据源URL配置
│
├── 📁 AI分析/
│   └── 内容分类.py                   # AI智能分类和总结模块
│
├── 📁 数据存储/
│   └── 数据库操作.py                 # 数据存储和读取模块
│
├── 📁 数据/
│   └── 新闻数据.json                 # 抓取的新闻数据（JSON格式）
│
└── 📁 .github/
    └── workflows/
        └── 每日抓取.yml              # GitHub Actions自动化配置

```

## 文件说明

### 📖 文档类

| 文件 | 用途 | 优先级 |
|------|------|--------|
| 快速开始.md | 新手5分钟快速上手 | ⭐⭐⭐ 必看 |
| 项目说明.md | 项目完整介绍和功能说明 | ⭐⭐ 推荐 |
| 部署指南.md | 详细部署教程和常见问题 | ⭐⭐ 推荐 |

### ⚙️ 配置类

| 文件 | 用途 | 需要修改 |
|------|------|---------|
| 配置文件.yaml | 所有系统配置 | ✅ 需要填入API Key |
| requirements.txt | Python依赖包 | ❌ 无需修改 |
| .gitignore | Git忽略规则 | ❌ 无需修改 |

### 🐍 Python程序

| 文件 | 用途 | 运行方式 |
|------|------|---------|
| 快速启动.py | 菜单式启动脚本 | `python 快速启动.py` |
| 主应用.py | Web界面主程序 | `streamlit run 主应用.py` |
| 新闻爬虫.py | 抓取新闻数据 | `python 数据抓取/新闻爬虫.py` |
| 内容分类.py | AI分析新闻 | `python AI分析/内容分类.py` |

### 📂 数据文件

| 文件 | 说明 |
|------|------|
| 数据/新闻数据.json | 存储所有抓取和分析后的新闻 |

## 使用流程

### 首次使用：
```
1. 安装依赖
   ↓
2. 配置API Key
   ↓
3. 运行快速启动.py
   ↓
4. 选择"完整运行"
   ↓
5. 浏览器自动打开界面
```

### 日常使用：
```
方式1: 运行快速启动.py → 选择"4.仅启动Web界面"
方式2: 直接运行 streamlit run 主应用.py
```

### 定期更新数据：
```
方式1: 手动运行快速启动.py → 选择"1.完整运行"
方式2: 设置GitHub Actions自动运行（推荐）
```

## 核心文件详解

### 🔧 配置文件.yaml
包含所有可配置项：
- AI服务配置（API Key在这里）
- 监控公司列表
- HR分类体系
- 爬虫参数
- 数据源配置

### 🚀 快速启动.py
交互式菜单脚本，提供：
- 依赖检查
- 配置验证
- 一键完整运行
- 单独运行各模块

### 🌐 主应用.py
Streamlit Web界面，功能：
- 首页概览展示
- 多维度筛选
- 统计分析图表
- 响应式设计

### 📰 新闻爬虫.py
数据抓取模块，特点：
- 支持多数据源
- 自动去重
- 时间过滤
- 反爬虫处理

### 🤖 内容分类.py
AI分析模块，功能：
- 判断是否HR相关
- 智能分类到HR模块
- 生成内容摘要
- 提取关键词

## 修改建议

### 🟢 可以修改：
- ✅ 配置文件.yaml 中的所有配置项
- ✅ 添加新的数据源（数据源配置.py）
- ✅ 调整HR分类（配置文件.yaml）
- ✅ 界面样式（主应用.py中的CSS）

### 🔴 不建议修改：
- ❌ requirements.txt（除非了解依赖关系）
- ❌ 核心爬虫逻辑（除非有爬虫经验）
- ❌ AI分析提示词（除非效果不好）
